# Connect Four Game
Connect Four game from freecodecamp Learn Python by Building five Games Course. 

## Tech Stack
- Python 

## Instructions
1. Clone this repo or download zip file. 
2. Open your command line. 
3. Run py connect.py to play the game. 
4. Move your cursor then click to drop the circles to connect four.
